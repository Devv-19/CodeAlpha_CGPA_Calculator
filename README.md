# CGPA Calculator 📚

This is a simple C++ program to calculate CGPA for a student based on course grades and credit hours.

### 🔧 Features:
- Take input for any number of courses
- Calculates total grade points
- Computes final CGPA
- Displays a clean CGPA report

### 🚀 Built With:
- C++
- Basic Loops & Arrays
- Console I/O
